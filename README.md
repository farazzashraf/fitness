# APEX Fitness Platform

APEX is a premium, high-performance fitness management platform designed for modern gym owners and their members. It features a sleek, dark-themed interface with advanced analytics and AI-driven business insights.

## 🚀 Features

### 👤 User Dashboard
- **Activity Tracking**: Monitor weekly workouts and calorie burn.
- **Goal Management**: Set and track fitness milestones.
- **Class Schedules**: Browse and book upcoming fitness sessions.
- **Subscription Management**: Easy access to Pro and Elite plan details.

### 🔐 Admin Dashboard (Command Center)
- **Business Overview**: Real-time tracking of total members, revenue growth, and churn rates.
- **AI Business Assistant**: An intelligent chatbot and insights engine that analyzes membership data to suggest business improvements and identify churn risks.
- **Customer Management**: Full list of Pro and Elite members with advanced search and filtering.
- **Analytics Engine**: Detailed traffic, signup, and conversion metrics powered by interactive charts.
- **Churn Target Monitoring**: Visual trends and proactive tools to maintain retention targets.

### 💳 Modern Checkout & Registration
- **Multi-step Registration**: Seamless onboarding flow for new members.
- **Tiered Pricing**: Clear distinction between Pro and Elite offerings.
- **Smart Checkout**: Dynamic plan selection based on user choice.

## 🛠️ Tech Stack

- **Frontend**: Next.js (App Router), React, TypeScript.
- **Styling**: Vanilla CSS with custom properties for high-performance glassmorphism and animations.
- **Visualization**: [Recharts](https://recharts.org/) for responsive, high-fidelity business analytics.
- **Icons**: Lucide-inspired custom SVG icons.
- **AI Integration**: Custom-built simulated AI logic for business intelligence.

## 📦 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📱 Responsiveness
The entire platform is built with a **Mobile-First** approach. The Admin Dashboard and AI Assistant are fully optimized for small screens, ensuring business owners can manage their gym on the go.
